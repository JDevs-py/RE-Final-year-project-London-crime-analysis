# RE-Final-year-project-London-crime-analysis
A data science project investigating crime patterns across London
This project aims to highlight trends and patterns in crimes that occured in london. Using publically available datasets provided by the Metropolitan police service, I was able to gain insights to the complexity of the problem crime poses in London, and gained a better understanding to how the dynamics of crime work.

This project was made possible with Excel, Power BI, Pandas and Google Colab.
4 datasets were used for this analysis.
- Borough level crime dataset featuring crimes that took place between January 2024 and December 2025
- Ward level crime dataset featuring crimes that took place between January 2024 and December 2025
- Average IMD score at Borough level
- Average IMD score at Ward level
